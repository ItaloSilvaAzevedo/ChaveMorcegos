# ChaveMorcegos
Chave dicotômica interativa para identificação de morcegos neotropicais! Ferramenta desenvolvida para ajudar pesquisadores, estudantes e entusiastas a identificar espécies de morcegos com base em características morfológicas.  Fácil de usar e guiá-lo-á passo a passo através de uma série de perguntas, levando-o à identificação correta da espécie.
